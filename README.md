# EcoBuild
O principal objetivo do EcoBuild é desenvolver uma plataforma online que permita a troca, venda e compra de materiais de construção usados, promovendo a sustentabilidade e a economia colaborativa. O objetivo é facilitar a reutilização de materiais, reduzindo o desperdício e os custos de construção para os usuários.

### Funcionalidades

- Cadastro e Login de Usuários:
    - Função: Permitir que os usuários criem uma conta e façam login.
- Cadastro de Materiais:
    - Função: Permitir que os usuários cadastrem materiais de construção que desejam **trocar** ou **vender**.
- Busca e Filtragem:
    - Função: Facilitar a busca por materiais específicos através de filtros por categoria, localização, etc. (Dentro da página de catálogo)
- Catálogo com trocas e compras:
    - Função: Implementar uma página que permita a negociação e troca entre usuários.
